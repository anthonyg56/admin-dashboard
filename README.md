## Getting Started

First, clone the repo and then to run the development server:

- Fork the repo
- Push to Vercel
- Create a postgres DB
- Add ENV variables
- attempt to sign in to seed the db
- Expand

```bash
npm i

# Then

npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
